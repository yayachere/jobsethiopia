# JobsEthiopia – Full-Stack Job Posting Platform

A modern job listing platform that allows users to browse, filter, and explore job opportunities through a responsive and intuitive interface.

🔗 Live Demo: https://jobsethiopia.vercel.app
🔗 GitHub Repository: https://github.com/yayachere/jobsethiopia  

---

## 📌 Overview

JobsEthiopia is a full-stack job board application built to simulate a real-world job listing platform.  
It focuses on dynamic filtering, API integration, responsive UI design, and clean component architecture.

This project demonstrates practical frontend development skills with real-world application structure.

---

## 🚀 Features

- 🔎 Dynamic job filtering and search
- 📄 Job detail page with structured layout
- 🔁 REST API integration
- 📱 Fully responsive design (mobile-first)
- ⚡ Optimized rendering using structured state management
- 🧩 Reusable React components
